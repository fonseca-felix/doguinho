# ☀️ Golden Dog 12:00

> "Onde o sol nunca se põe para um novo melhor amigo."

O **Golden Dog 12:00** é uma aplicação web minimalista e interativa feita com Flask. O objetivo é simples: nomeie seu novo amigo e receba uma foto aleatória de um cãozinho, tudo isso acompanhado de uma experiência visual fluida com elementos que reagem ao seu mouse.

---

## ✨ Destaques do Projeto

* **Busca em Tempo Real**: Integração direta com a *The Dog API* para garantir que você sempre conheça um cão novo.
* **Interface Imersiva**: Cursor personalizado em formato de sol e uma bolinha de tênis animada via Canvas que persegue o seu mouse.
* **Feedback Lúdico**: Mensagens de erro e estado do sistema personalizadas, mantendo o tom "solar" do projeto.
